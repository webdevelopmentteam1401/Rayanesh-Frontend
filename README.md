# Rayanesh Site React.js Frontend
#### Whole Project Structure
* Backend: Django
* Frontend: React.js
* Database: Postgres
* Deployment: Docker and Docker Compose

# Installation
1. Clone the repository: 
```
git clone https://github.com/webdevelopmentteam1401/Rayanesh-Frontend.git
```
2.
```
yarn
```
3.
```
yarn start
```
